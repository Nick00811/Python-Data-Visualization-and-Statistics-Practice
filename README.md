Python Data Visualization and Statistics Practice
Overview

This repository contains a collection of Python practice exercises focused on data visualization, statistics, and exploratory data analysis (EDA). The project is designed to strengthen foundational data science skills through hands-on coding and visual interpretation of data.

Objectives

Practice core statistical concepts using Python

Perform exploratory data analysis (EDA) on datasets

Visualize data using different charts and plots

Understand relationships between variables using correlation and regression

Implement basic probability and statistical simulations
